# uber-powerbi-dashboard
End-to-end Power BI project using Uber ride data, featuring data cleaning, DAX measures, and interactive dashboards for business insights.


This project showcases an interactive Power BI report built using Uber ride data.
The dashboard provides insights into trip volume, revenue trends, peak demand hours,
and overall performance to support data-driven decision-making.

### Tools Used
- Power BI
- DAX
- Data Cleaning & Transformation

### Key Insights
- Trip trends by time and location
- Revenue and demand analysis
- Peak hours and ride patterns
